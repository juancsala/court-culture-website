# Court Culture

> More than a sport. A way to connect.

Court Culture is a social tennis community based in Monterrey, México. A space where tennis is the excuse to play, meet people, and build real connections.

## Tech Stack

- **React + TypeScript** — Frontend
- **Vite** — Build tool
- **Tailwind CSS** — Styling
- **Framer Motion** — Animations
- **React Three Fiber / Three.js** — 3D elements

## Getting Started

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
```

Deploys to **Vercel**. Build output is in `dist/`.
